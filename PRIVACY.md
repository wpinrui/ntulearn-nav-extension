# Privacy Policy

**NTUitive** does not collect, store, or transmit any personal data.

All user preferences (feature toggles, course list cache) are stored locally in your browser using the `chrome.storage` API and are never sent to any external server.

This extension does not use analytics, tracking, or any third-party services.

If you have questions, open an issue at https://github.com/wpinrui/ntuitive/issues.
